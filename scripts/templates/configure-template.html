Letters and Numbers

This is a game to familiarise people with a keyboard, it is aimed at children and supports the 26 letters of the english alphabet and numbers 0 to 9. The application will display a keyboard key and the player must press the correct key to allow the rocket ship to travel up the page. A score is shown when the ship reaches the top. The engine attempts to favour those characters that have been incorrectly guessed; all played keys are shown in a score page once the rocket has reached the top of the page.

The game is hosted on github (https://github.com/bazwilliams/bazapp)

Configure your game below, the duragadget theme is styled similar to Duragadget's child proof keyboard, the black keys are themed based on a macbook keyboard. The currently displayed key can be spoken if configured below when each is displayed; clicking the key will repeat the phrase. The game length slider adjusts how many steps the rocket must take to complete the game.
